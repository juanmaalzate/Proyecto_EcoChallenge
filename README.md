# EcoChallenge

**EcoChallenge** es un videojuego educativo hecho en **Python** usando **Pygame**.  
Su objetivo es enseñar a los niños de primaria a clasificar correctamente los residuos mediante el juego.



# ¿Cómo funciona?

- Los residuos (figuras geométricas) caen desde la parte superior de la pantalla.  
- El jugador usa las flechas **← y →** para mover los residuos y clasificarlos en las canecas.  
- Cada acierto suma puntos y cada error quita vidas.  
- La velocidad aumenta con el tiempo para hacerlo más desafiante.



# Cómo ejecutarlo

1. Abre una consola o terminal.  
2. Asegúrate de tener instalado **Python** y **Pygame**:  
   ```bash
   pip install pygame
````

3. Ejecuta el juego con:

   ```bash
   python ecochallenge.py
   ```



# Objetivo del proyecto

Fomentar la educación ambiental mediante la gamificación, enseñando a los niños la correcta clasificación de residuos reciclables y no reciclables.
